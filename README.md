# Voitkevics_UnityUI

## Par spēli - Bikini Bottom Stilu Skapis

Bikini Bottom Stilu Skapis ir tēlu apģērbšanas spēle, kas ļauj lietotājam apģērbt Sūkļa Boba Kvadrātbikša un Patrika Zvaigznes izskatu. Spēle piedāvā plašu apģērbu un aksesuāru kolekciju, dinamisku UI dizainu un tēliem atbilstošu vizuālo astētiku.

### Izdarīto darbu saraksts:
- [x] Fona mūzika
  - [x] 3 dziesmas "Main Menu" ainā (Shuffle/Jauktais atskaņošanas režīms)
  - [x] 5 dziesmas "Game" ainā (Shuffle/Jauktais atskaņošanas režīms)
- [x] Skaņas efekti
  - [x] Toggle elementu skaņa
  - [x] Slider elementu skaņa
  - [x] Skaņu efekti, kad tiek izvēlēts tēls no  DropDown paneļa
  - [x] Skaņas efekti, kad uzspiež uz tēla
- [x] Ainu pārslēgšana
- [x] Dinamisks UI dizains
  - [x] UI vizuālā astētika mainās atkarībā no izvēlētā tēla
  - [x] Izmainās teksta izvades lauks
- [x] Izveidoti tēli
  - [x] Sūklis Bobs Kvadrātbiksis
    - [x] Kategorijas: Amuleti, Bikses, Cepures, Kurpes (3 apģērbu gabali katrā)
    - [x] Personalizēts apraksts Scroll View logā
  - [x] Patriks Zvaigzne
    - [x] Kategorijas: Krekli, Bikses, Cepures, Kurpes (3 apģērbu gabali katrā)
    - [x] Personalizēts apraksts Scroll View logā
- [x] Spēles darbība
  - [x] Drag & Drop princips apģērbu uzlikšanai
  - [x] Slaideri tēla augstuma un platuma mainīšanai
  - [x] Slaideri drēbju platuma un augstuma mainīšanai (Slaideri maina izmēru tikai tām drēbēm, kuras ir izvilktas ārā no skapja, un, kura drēbju kategorija šobrīd ir aktīva (izvēlēta)
  - [x] Vārda ievade (Nedrīkst būt cipars vārdā, ne garāks par 15 burtiem) 
  - [x] Dzimšanas gada ievade (Minimums: 1926. gads)
  - [x] Teksta izvades maiņa atkarībā no izvēlētā tēla
